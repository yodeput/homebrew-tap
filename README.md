# Homebrew Tap

This tap contains the **rslink** formula.

## Installation

```bash
brew tap yodeput/tap
brew install rslink
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [rslink](Formula/rslink.rb) | Copy/move folders with rsync and create symlinks |

## Update

```bash
brew upgrade
```

## Uninstall

```bash
brew uninstall rslink
brew untap yodeput/tap
```
